# My-Flowchart-Site

This project contains the workflow write-up you use at the start of new projects, now organized as a foundation for an interactive dashboard tool.

## Project structure

- `/index.html` — main page and current flowchart UI
- `/assets/css/styles.css` — shared styling
- `/assets/js/tailwind.config.js` — Tailwind CDN configuration
- `/assets/js/app.js` — JavaScript entry point for upcoming interactivity

## Next evolution

The current structure separates content, styles, and behavior so future dashboard features can be added without keeping everything in one HTML file.
